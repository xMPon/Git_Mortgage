# Git_Mortgage

A simple Python project for mortgage calculations, payment schedules, and interactive financial graphs.

## Features
- Calculate mortgage payments and generate amortisation schedules
- Visualise loan repayment, house equity, and forecasted house prices
- Web interface (Flask) to display interactive graphs in your browser

## Requirements
- Python 3.8+
- pandas, numpy, matplotlib, numpy-financial, flask

## Usage
   - Run calculations and view plots:
     ```bash
     python house_calculations.py
     ```
## Project Structure
- `house_calculations.py` — Main mortgage logic and plotting
- `.gitignore` — Ignores Python cache files
